# 🍽️ Suneel SmartServe – Digital Restaurant Platform

A modern and responsive restaurant website designed to help businesses go online, showcase their menu, and interact with customers easily.

## 💡 Project Overview
Suneel SmartServe is a frontend web application built using HTML, CSS, and JavaScript.  
It provides a clean and user-friendly interface for restaurant businesses to display their services and improve customer engagement.

---

## ✨ Features

- 🔥 Modern and responsive UI design
- 🍽️ Menu section with filtering (Veg / Non-Veg)
- 🖼️ Image gallery for food showcase
- 📊 Business stats (orders, ratings, chefs)
- 🌙 Dark mode toggle
- ⭐ Customer review & rating system (local storage)
- 💬 WhatsApp integration for direct ordering
- 📍 Google Maps location integration

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage (for storing reviews)
